# Mimi

**Aesthetic intelligence system—mapping taste, generating zines, and shaping cultural positioning for the algorithmic era.**

---

## What is Mimi?

Mimi is not a design tool.  
It is a system for **understanding and operationalizing taste**.

Where traditional tools help you *create*, Mimi helps you:
- observe your aesthetic patterns  
- map your taste as signal  
- translate intuition into structured output  

The result: **editorial artifacts, strategic insight, and cultural positioning**.

---

## Core Concept

Mimi operates on a simple idea:

> your taste is already a system — it just isn’t visible yet

Every reference, save, and instinct forms a pattern.  
Mimi captures that pattern and turns it into:

- **Zines** → expressive outputs  
- **Signals** → structured taste data  
- **Positioning** → directional strategy  

---

## System Layers

### ✦ Tailor (Identity Layer)
Defines your aesthetic DNA:
- silhouettes  
- textures  
- references  
- emotional tone  

This is not a moodboard.  
It is a **compressed identity model**.

---

### ✦ Thimble (Observation Layer)
Captures what you naturally gravitate toward:
- saved references  
- links  
- visual inputs  

Transforms behavior into:
> measurable aesthetic signal

---

### ✦ Studio (Creation Layer)
Generates outputs from your system:
- zines  
- visual concepts  
- editorial narratives  

Creation becomes:
> a consequence of clarity, not randomness

---

## Running Locally

```bash

```

```bash
GEMINI_API_KEY=your_key_here
```
